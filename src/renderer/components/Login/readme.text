一、登录：
1、记住密码：
    若勾选，首先把是否记住密码状态记录localStorage。
        登录成功时，把当前登录密码记录localStorage。
        再次进入登录页面时，若记住密码为true，自动填充密码。
    若取消勾选，首先把是否记住密码状态记录localStorage。
        把localStorage中已经记住的密码清空。
2、用户名下拉框：
    登录成功后，会把当前用户记录进localStorage，下次再进入可以在下拉框中直接选择。
    可以删除，会把localStorage中相应用户名删除。
3、加密/解密：传数据/取数据都通过 AES 加密。
